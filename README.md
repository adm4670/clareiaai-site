# Clareia.ai Site

Landing page moderna e responsiva para a Clareia.ai — startup educacional que utiliza IA para preparação personalizada para o ENEM.

## Estrutura do Projeto

```
clareiaai-site/
├── index.html                  # Página principal (landing page)
├── css/
│   └── style.css              # Estilos com identidade visual da marca
├── js/
│   └── main.js                # JavaScript (animações, formulário, navegação)
├── images/
│   ├── logo.png               # Logo oficial da Clareia.ai
│   ├── hero-illustration.svg  # Ilustração conceitual do hero
│   ├── feature-*.svg          # Ícones de funcionalidades (6 ícones)
│   └── step-*.svg             # Ilustrações das etapas (3 passos)
└── material-de-apoio/
    ├── logo.png               # Logo original
    └── clareia_ai_documentacao.md  # Documentação de conteúdo
```

## Identidade Visual (extraída da logo)

| Cor | Código | Uso |
|-----|--------|-----|
| Azul Primary | `#0070ab` | Elementos principais, botões, links |
| Azul Escuro | `#232f3f` | Textos, footer |
| Rosa/Magenta | `#da007e` | Detalhes, destaques, hover states |
| Laranja | `#fe4e19` | Alertas, badges, detalhes secundários |

## Seções da Landing Page

1. **Hero** — Proposta de valor principal com CTA
2. **Problema/Solução** — Dores do estudante vs. solução Clareia.ai
3. **Proposta de Valor** — 3 pilares da plataforma
4. **Funcionalidades** — 6 cards com recursos do produto
5. **Como Funciona** — 3 passos da jornada do usuário
6. **Lista de Espera** — Formulário de captura de leads
7. **Footer** — Links e informações de contato

## Como visualizar

Abra o arquivo `index.html` em qualquer navegador moderno.
